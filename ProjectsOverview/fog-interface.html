Project 3 · Automotive HCI

Fog Distance Perception Interface

Enhancing driver distance perception in foggy conditions using dynamic visual cues when natural visibility breaks down.

Problem

In foggy conditions, drivers lose reliable depth cues. Existing dashboards do not compensate for reduced distance perception, increasing accident risk.

Research Question

How can visual distance cues in automotive interfaces improve driver understanding of surrounding vehicles in fog?

Approach

A controlled lab study comparing driving with and without visual distance cues using simulated fog scenarios.

Participants

N = 18

Conditions

Visual vs No Visual

Tasks

Emergency scenarios

Metrics

RT, Errors, UEQ

Key Outcome

While visual cues showed promising behavioral trends, the statistical analysis did not reach significance (t-test = 0.16). The work contributes design insights rather than performance guarantees.
