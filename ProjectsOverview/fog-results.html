Fog Interface — Results

Effect

Faster behavioral response trends

Statistics

Not statistically significant

t-test, p > .05

Meaning

Design signal, not performance proof

With Visual Distance Cues

Interface Strategy

Abstract distance bars + vehicle-centric icon encode relative proximity without numeric precision.

Without Visual Cues

Baseline Strategy

Driver relies exclusively on degraded natural vision under fog conditions.

Scientific Interpretation

Although statistical significance was not achieved, the observed trends indicate perceptual and behavioral benefits of abstract distance encoding in degraded visual conditions.

Design Implication

Abstraction is suitable for perception support, even when performance gains are not statistically significant.

Supports safety-oriented interface design logic rather than performance-only optimization.
