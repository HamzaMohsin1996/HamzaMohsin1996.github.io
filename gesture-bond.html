Enhancing Social Presence through Gesture-Based Interactions

View Prototype
Overview
Problem

In remote collaborative learning, traditional text-based chat systems often fail to adequately support social presence and emotional connectivity. This project investigates how gesture-based avatar systems can enhance these aspects, thereby improving collaboration in remote educational settings.

Outcome

Gesture Bond significantly enhanced remote communication by using gesture-based avatars, leading to increased social presence, emotional connectivity, and user engagement compared to traditional text-based systems. Users reported feeling more naturally connected, effectively improving the quality of remote collaboration.

Project Overview
GestureBond

A gesture-based avatar communication tool designed to enhance emotional connectivity and social presence in remote interactions. Developed as part of the “Natural User Interface” course, this project leverages advanced UX research and technical innovation to create a more immersive and engaging communication experience.

Gesture-based avatar systems lead to higher social presence and emotional connectivity than text-based chat systems.
User Testing and Iterative Design

We conducted two rounds of user studies to evaluate and enhance the Gesture Bond interface. The objective was to measure user satisfaction, ease of use, and social presence.

First User Study

Objectives: To evaluate the initial gesture-based interface.

Methodology: Participants completed questionnaires and participated in interviews after using the interface.

Key Findings
  • Users found the gesture-based interface innovative but had difficulty with gesture recognition.
  • Some gestures were not intuitive and required better visual cues.
  • Participants appreciated the concept of silent communication in quiet environments.
User Study Results

Enhancements Based on First Study

Based on the feedback from the first user study, we made the following enhancements:

Enhancements
  • Added digital onboarding to help users get started with the system.
  • Implemented a beep notification to alert users without requiring constant screen attention.
  • Enlarged the avatar of the second person for better visibility.
  • Added name labels below avatars for better identification of users.

Final User Study

Objectives: To evaluate the improved gesture-based interface.

Methodology: Participants used the enhanced interface and were interviewed for feedback.

Key Findings
  • Users found the enhanced gesture recognition significantly better.
  • Visual cues and onboarding helped in understanding and performing gestures correctly.
  • Overall user satisfaction and social presence improved significantly.

Final Results and Conclusion

The iterative design process helped in significantly improving the Gesture Bond interface. The final user study showed higher user satisfaction, better ease of use, and enhanced social presence.

Key Takeaways and Future Directions
The Importance of User Feedback

Iterative testing and continuous feedback from users were crucial in identifying and addressing pain points, leading to a more user-friendly and effective interface.

Simplicity is Key

Complex features can often overwhelm users. Simplifying interactions and focusing on core functionalities greatly improved user satisfaction and usability.

Effective Communication

Clear and non-intrusive communication methods, such as sound notifications and visual cues, enhanced user engagement and social presence.

Adaptability

Flexibility in design and the ability to adapt based on user feedback is essential for creating a successful user experience.

Future Improvements
  • The trained dataset can be broadened to create more accurate gesture recognition.
  • Needs more 3D effect for each character.
  • Customization should be possible like personalized avatars.
  • Addition of more signs if required for the users.
